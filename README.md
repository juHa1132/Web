# Professional Web Project – Portfolio Presentation

This repository contains a professional website project originally developed for **Aleksandra Adamiec**, a sports psychologist. The project was created as a portfolio showcase and was **never deployed to production** on the internet.

The repository is shared on my GitHub profile with the author's permission and serves purely as a demonstration of my web development work.

## 🧠 About the Project

- A modern, responsive website designed for a sports psychologist  
- Focus on clean UI/UX and professional presentation  
- Includes a contact form with backend support  
- Built as part of my portfolio of web development projects  

## 🛠️ Tech Stack

- **Frontend:** React (Create React App)  
- **Backend:** PHP (used for handling the contact form)  
- **Styling & UI:** Custom CSS / component-based architecture  
- **Architecture:** Client–server communication via HTTP  

## 🚀 How to Run the Project Locally
git clone <repository-url>
cd <project-folder>
npm install
npm start

Then open:

http://localhost:3000
